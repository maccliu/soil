# Soil

Soil是一套PHP底层框架，基于MIT协议。

Soil的英文意思是土地、泥土。正如Soil的名字一样，PHP的底层框架实现起来非常肮脏，需要对各种PHP的基础知识和编程概念都搞得相当门清，各种接口规范和调用关系也要了然于胸。实现PHP底层框架虽然非常艰难，不过实现后，的确能为应用代码带来相当大的便利性。
